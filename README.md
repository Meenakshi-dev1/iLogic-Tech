# iLogic Tech – Company Website

Static website for iLogic Tech, an integrated technology solutions company with offices in Singapore and India (Pondicherry).

## Pages
- **index.html** – Home page (Singapore view)
- **service.html** – Service detail pages (mobile apps, software, web, AI/ML, networking, cybersecurity, CCTV)
- **solution.html** – Business solutions (POS, ERP, CRM, CMS, Dine App)
- **project.html** – Hardware products
- **exam.html** – Exam centre (LanguageCert, CELPIP, BeingCert)
- **languagecert.html** – LanguageCert detail page
- **career.html** – Careers
- **internship.html** – Internship programme
- **sitemap.xml** – Sitemap for Google Search Console

## Tech Stack
Plain HTML + CSS + JavaScript (no frameworks).

## SEO
- Meta titles and descriptions are set per-page in HTML
- service.html injects dynamic meta via script.js based on `?slug=` URL param
- JSON-LD structured data included in service.html
- sitemap.xml submitted to Google Search Console at search.google.com/search-console
